Drupal Console Spanish language package
===

The [Drupal Console](http://drupalconsole.com) project since 1.0.0 stable version, languages are separated from DrupalConsole installer.

This package is a re-distribution of the effort of community to provide a translation for Spanish speakers.

# How to install

Located in the root of your Drupal

```
$ composer install drupal/console-spanish
```
